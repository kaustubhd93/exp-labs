Headings

Headings define the outline of your site. There are 6 levels of headings.

Lists

This is how you make an unordered list:

  • Add a "ul" element in a block format.
  • Then add an "li" element inside the "ul" block
  • No element needs to be in any particular order.

    • Adding a paragraph in a "li" element. As you can see this does not work. Only "li" is allowed inside the "ul" block

  • Adding a paragraph in a "li" element. This works because the paragraph is wrapped inside a "li" element.

This is how you make an ordered list:

  1. Create a "ol" block and add a "li" element inside it.
  2. No need to add numbers, the serial no for next element will auto increment.
